# Sentinel Evolution

This directory contains snapshots of Sentinel at key development milestones.

## Snapshots

### Month 1: The Foundation (August 2025)
Single-file prototype. Basic screen capture and Flask server.

### Month 3: The Refactor (October 2025)  
Modular architecture. TLS support. Separate Swift classes.

### Month 5: Production (December 2025)
Full-featured release. Built-in HTTP server. Test suite.

Each snapshot is a working version you can build and run.
